Robo Drive

A hand-following car bot that uses an IR sensor to detect obstacles or hand gestures. When a hand is placed in front, the bot moves forward, mimicking motion sensing. It’s a simple yet smart example of sensor-based automation.

VoxaLens

An AI-powered smart lens that converts spoken words into text in real-time using speech-to-text technology. Designed to help people with hearing disabilities or in noisy environments, it makes communication more accessible and futuristic.

Inkscape Reality

A physical sketching machine that turns real images into pencil sketches using Inkscape extensions. The system processes the image digitally and then draws it out on paper, blending software precision with hardware creativity.

LazerGaze Bot

An intelligent bot that tracks human face movements in real-time and responds by rotating in the same direction. It also uses a laser pointer that aligns with hand gestures to highlight objects or locations, making it useful for presentations, assistive tech, and futuristic interactions.

H-Trak

H-Trak is a human motion tracking project that uses computer vision to capture and simulate real-time body movements. A 3D human figure is created in Blender, which mirrors hand gestures, face expressions, and full-body poses with high accuracy

AI Photo Booth

A smart photo booth that captures real-time images and transforms them into different artistic styles using AI. Users can choose from various effects like cartoon, sketch, or futuristic looks, and the system instantly generates the styled version. It’s a fun and creative fusion of photography and AI image generation.

Sky Rover

A custom-built drone designed for smooth and stable flight, perfect for capturing cinematic aerial videos. It uses high-resolution camera support and intelligent flight controls to shoot dynamic shots from the sky, making it ideal for filmmaking, exploration, and landscape photography.

Memory Matrix

A fun and challenging Python-based game built on Trinket that tests your memory skills. Players have to memorise and repeat randomly generated patterns within a grid. As the levels go up, the difficulty increases, making it a perfect brain-training and focus-boosting game.

AI Rock-Paper-Scissor

An interactive Rock-Paper-Scissors game where an AI bot plays against a human using computer vision. A camera captures the player’s hand gesture, and the AI responds with its move using a physical model powered by Arduino and motors. It’s a cool mix of gameplay, robotics, and machine learning.

Guide नेत्र

A voice-assisted app built using MIT App Inventor, specially designed for visually impaired users. It detects nearby obstacles and alerts the user through audio feedback. The app also features an SOS button for making instant emergency calls, offering both safety and independence to those with visual disabilities.

Bio Blueprint

A 3D animated model created in Blender that showcases a futuristic, sustainable city design. It highlights eco-friendly structures, renewable energy setups, and green spaces through smooth animations. The project blends creativity and environmental awareness to promote sustainable living through visual storytelling.

Space Shooter

A classic arcade-style game made using Python and Pygame, where players control a spaceship using arrow keys. The goal is to shoot down enemies and dodge incoming attacks in a fast-paced space battle. It's a fun and interactive way to learn game development and logic building.

Compliment Cloud

A feel-good HTML-based webpage that takes the user’s name as input and returns sweet, personalised compliments. It’s designed to brighten someone's day with kind words, all served through a simple and fun web interface. Great for spreading positivity with just a click!
